OpenShift Integration

Connect to Red Hat OpenShift clusters, bringing enterprise-grade container orchestration capabilities to our platform.

Cluster Health Check Report

Generate and download comprehensive reports for individual clusters, including performance metrics, health status, and resource utilization.

Kube-bench Integration (CIS Compliance Reports)

AI Platform Integrations

Integration with Local AI & other AI platforms including AWS Bedrock, DeepSeek, and Claude, bringing advanced analytics and intelligence to your infrastructure.

Anonymization & Redaction

Automatically redacts or masks sensitive data in input data that is shared with the AI model to ensure privacy and data compliance.

Edit Resource YAML File

Modify and apply YAML configuration files for clusters, managing settings such as performance parameters, resource configurations etc.

Trivy Integration for Manifest Analysis

Performs static security scanning of Kubernetes manifests to detect misconfigurations and vulnerabilities using Trivy.

Q4 2025

SSO Integration

Enables seamless and secure authentication via Single Sign-On, improving user access control and management.

Helm Marketplace

A comprehensive marketplace for Helm packages, allowing easy discovery, deployment, and management of applications across your Kubernetes clusters.
